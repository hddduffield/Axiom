## TRIGGERING CONDITIONS

### Structured logic
```
TRIGGER if ANY of:
  - Existing irrevocable trust contains outdated provisions (old age limits, restrictive distribution standards, named individuals no longer appropriate)
  - Beneficiary needs vary from original trust intent
  - State law changes have created opportunity for improvement (e.g., dynasty extension)
  - GST allocation imperfect and decanting can fix
```

### Natural-language explanation
"Decant" the assets of an existing irrevocable trust into a new trust with updated terms, under state decanting statutes. Allows modification of "irrevocable" trusts in defined ways without court process.

### Hard disqualifiers
- Trust language explicitly prohibits decanting
- State law doesn't permit needed modification
- Beneficiary objections without resolution

---

## WHAT IT IS

Decanting is the trustee's exercise of discretionary distribution power to "pour" trust assets from an existing trust into a new trust with different terms. State decanting statutes (Georgia: O.C.G.A. §53-12-62 [VERIFY 2026]) define what modifications are permitted. Some states (Delaware, South Dakota, Nevada) have very flexible statutes; Georgia is more restrictive.

Common uses:
- Extend trust duration (when state RAP allows)
- Update distribution standards
- Change trustee provisions
- Fix administrative errors
- Update GST exemption allocation

---

## WHY WE RECOMMEND IT

When an irrevocable trust no longer serves the family's intent — perhaps drafted under different state law, different family circumstances, different tax law — decanting can update terms without going to court (which would be slow, expensive, public). For trusts decades old, this can be transformative.

## QUANTIFIED IMPACT FRAMEWORK
- Highly fact-specific
- Avoids court costs ($10K-$50K+ for trust modification proceedings)
- Can capture material tax planning opportunities (GST fixes, situs changes)

---

## IMPLEMENTATION STEPS

1. Trustee + attorney analyze decanting authority under state law
2. Beneficiary notification per state requirements
3. New trust drafted with desired terms
4. Trustee exercises decanting power; assets transfer
5. Old trust terminates; new trust succeeds
6. Tax filings reflect transition
