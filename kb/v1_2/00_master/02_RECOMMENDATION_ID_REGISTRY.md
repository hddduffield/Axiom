# RECOMMENDATION ID REGISTRY

Every recommendation in the Axiom Knowledge Base. IDs are stable — once assigned, they never change. New recommendations append; deprecated stay registered.

**Status legend:**
- **Active** — standard recommendation, included whenever triggered
- **Active-Cautioned** — included when triggered but with extra senior-advisor review
- **Landmine** — only included with explicit senior-advisor direction; default OFF
- **Advanced** — appropriate but requires specialist counsel beyond firm's in-house bench
- **Deprecated** — kept in registry; not used in current generation

**Archetype legend:**
- **PRE** — Pre-Exit Business Owner
- **POST** — Post-Exit Business Owner  
- **ACT** — Active Owner Without Exit Horizon
- **FO** — Family Office / Multi-Gen
- **FOUND** — Pre-Liquidity Founder
